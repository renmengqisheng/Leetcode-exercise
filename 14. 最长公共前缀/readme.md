编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

示例 1:

输入: ["flower","flow","flight"]
输出: "fl"

示例 2:

输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。

说明:

所有输入只包含小写字母 a-z 。

思路：

首先找到所有字符串的最短长度，然后以最短长度作为公共前缀的可能最长值，对所有字符串的每一列相应位置进行比较，如果都相等，则加入结果字符串中，如果某一列不是全相等，则直接返回保存的结果字符串。时间复杂度为O(m * n)，额外空间复杂度为O(1)，其中m为所有字符串的最短长度，n为字符串数组的个数。
